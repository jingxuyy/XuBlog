

# Blog

### 技术栈

- SpringBoot
- MybatisPlus
- SpringSecurity
- EasyExcel
- Swagger2
- redis
- Echarts
- Vue
- ElementUI
- MySQL

### 介绍

项目总体可以分为两个部分：1. 博客前台系统  2. 博客后台系统

### sg-blog-vue 为前台系统的前端源代码

### sg-vue-admin 为后台系统的前端源代码

### 其它为后端源代码
